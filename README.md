# 📰 News App (React)

A responsive and dynamic News Application built using **React.js** that fetches real-time news from an external API and displays it in a clean and user-friendly interface.

---

## 🚀 Features

- 🌍 Fetches **latest news** from API
- 🗂️ Category-based news (Business, Sports, Technology, Health, etc.)
- 🔄 **Infinite scrolling / Pagination**
- ⚡ Fast and responsive UI
- 🔗 Dynamic routing using React Router
- 📱 Mobile-friendly design

---

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Language:** JavaScript (ES6+)
- **Styling:** CSS 
- **API:** News API (REST)
- **Routing:** React Router
